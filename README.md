# osfive

Osfive is an RTOS/SDK for microcontrollers.

## Key features
- Single-threaded
- Application-centric
- Tickless operation

## Supported libraries
- BSD libc library
- libFLAC

## Supported platforms
1. ARM
..* Maxim MAX32625
..* Maxim MAX32660
2. MIPS
..* Microchip PIC32
..* Nordic Semiconductor nRF52
..* STM32
3. RISC-V
..* SiFive E300G
..* Microsemi
