# osfive

Osfive is an RTOS/SDK for microcontrollers.

## Key features
- Single-threaded
- Application-centric
- Tickless operation

## Supported libraries
- BSD libc library
- libFLAC

## Supported platforms
- RISC-V
..* Microsemi
..* SiFive E300G
- ARM
..* Maxim MAX32625
..* Maxim MAX32660
- MIPS
..* Microchip PIC32
..* Nordic Semiconductor nRF52
..* STM32
