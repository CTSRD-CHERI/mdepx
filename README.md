# osfive

Osfive is an RTOS/SDK for microcontrollers.

## Key features
- Single-threaded
- Application-centric
- Tickless operation

## Supported libraries
- BSD libc library
- libaeabi
- libFLAC
- md5
- tinycrypt
- xz

## Supported platforms
- ARM
  * Maxim MAX32625
  * Maxim MAX32660
  * Nordic Semiconductor nRF52
  * STM32
- MIPS
  * Microchip PIC32
  * BERI
- RISC-V
  * Microsemi
  * SiFive E300G
