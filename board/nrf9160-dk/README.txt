This is an initialization code for nRF9160-DK board.

A complete LTE-M/NB-IoT and GPS demo application could be found here:
https://github.com/machdep/nrf9160

For GPS ensure your board is version 0.9.0 and upper.

Ensure that the modem firmware is latest version.

Use external GPS antenna that comes in the package (on-board switch SW12 has to be in the CON position).

Put the board as close as possible to a window or use it under open space.
